linux打不开vlc怎么办下面是解决方法

发表于2020-01-23|更新于2021-12-25|问题解决
|阅读量:

今天我发现打不开vlc了我就百度了一下发现修改下面的就可以完美解决这个问题

打开vlc出现
root@kali:~# vlc
is not supposed to be run as root. Sorry.
If you need to use real-time priorities and/or privileged TCP ports
you can use vlc-wrapper (make sure it is Set-UID root and
cannot be run by non-trusted users first).


或者打击vlc图标没有反映
修改vi /usr/bin/vlc文件即可解决问题
叫下面的geteuid修改为getppid
