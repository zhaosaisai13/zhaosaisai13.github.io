常见的函数

函数 作用
phpinfo()函数 查看php关于PHP的信息
strlen()函数 函数返回字符串的长度，以字符计
isset()函数 函数用于检测变量是否已设置并且非 NULL
count() 函数 用于返回数组的长度（元素数）
end()函数 输出数组中的当前元素和最后一个元素的值
current() end()函数相反输出数组中的当前元素和最前面一个元素的值
explode()函数 把字符串打散为数组
trim()函数 函数移除字符串两侧的空白字符或其他预定义字符
str_ireplace()函数 函数替换字符串中的一些字符（不区分大小写）
strrchr()函数 查询字符串最后的要找到内容，并获取后面的内容
strrpos()函数 字符串中最后一次出现的位置
substr()函数 函数指定返回字符串的一部分
getimagesize()函数 函数用于获取图像及相关信息返回是一个数组
image_type_to_extension()函数 根据指定的数字图像类型返回对应的后缀名

substr()

substr()函数返回字符串的一部分

参数 描述
参数1 必需。规定要返回其中一部分的字符串。
参数2 必需。规定在字符串的何处开始。正数 - 在字符串的指定位置开始负数 - 在从字符串结尾开始的指定位置开始0 - 在字符串中的第一个字符处开始
参数3 可选。规定被返回字符串的长度。默认是直到字符串的结尾。正数 - 从 start 参数所在的位置返回的长度负数 - 从字符串末端返回的长度

image_type_to_extension()根据指定的数字图像类型返回对应的后缀名
